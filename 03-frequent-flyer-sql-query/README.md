# Frequent Flyer SQL Query Analysis

## Overview
This project documents the development of a SQL query to generate a distinct list of frequent flyer names based on trip records within a specific date range. The project focused on joining related tables, filtering records by travel date, and returning clean query results from a relational database.

## Tools Used
- MySQL Workbench
- SQL

## What I Did
- Reviewed the frequent flyer database structure
- Identified the tables needed for the query
- Joined flyer and trip-related data using shared keys
- Applied a date range filter to limit records
- Used `DISTINCT` to return unique frequent flyer names
- Verified the query output in MySQL Workbench

## Skills Demonstrated
- SQL query writing
- Table joins
- Date filtering
- DISTINCT selection
- Relational database analysis
- Query result verification
