# Sales Database ERD Design

## Overview
This project documents the design of an entity relationship diagram for a hard drive wholesale sales business. The work focused on analyzing a sales form, identifying the main entities involved in the business process, and structuring the relationships needed to support accurate database design.

## Tools Used
- ERD modeling
- Database design concepts
- Relational database analysis

## What I Did
- Reviewed the sales form and business scenario
- Identified key entities involved in the sales process
- Determined important attributes for each entity
- Defined relationships between customers, orders, products, and related records
- Organized the structure into a relational database design model

## Skills Demonstrated
- Entity relationship diagram design
- Database planning
- Business process analysis
- Relational data modeling
- Attribute and relationship identification
