# Practice Database Table Creation

## Overview
This project documents the creation of two related tables in a MySQL practice database: `Employee` and `Office`. The work focused on defining appropriate data types, establishing a foreign key relationship, and verifying the table structure and inserted records in MySQL Workbench.

## Tools Used
- MySQL Workbench
- SQL

## What I Did
- Created an empty database named `practice`
- Built an `Office` table with office location data
- Built an `Employee` table with employee data
- Defined `Office_ID` as a foreign key linking the employee table to the office table
- Applied reasonable data type constraints to fields
- Inserted sample records into both tables
- Verified the code and schema structure in MySQL Workbench

## Skills Demonstrated
- SQL table creation
- Primary key and foreign key design
- Relational database structure
- Data type selection
- Record insertion
- Schema verification in MySQL Workbench
