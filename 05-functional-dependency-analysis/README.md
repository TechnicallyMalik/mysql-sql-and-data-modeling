# ERD Functional Dependency Analysis

## Overview
This project focused on interpreting an entity relationship diagram and identifying the functional dependencies implied by the structure. The work emphasized relational database theory, key relationships, and how entity participation affects dependency patterns in a database design.

## Tools Used
- ERD interpretation
- Relational database theory
- Functional dependency analysis

## What I Did
- Reviewed the ERD structure and entity relationships
- Identified the keys associated with the entities
- Analyzed how the ternary relationship affected dependency logic
- Determined which attributes and entity combinations functionally depended on others
- Interpreted how the relationship structure influenced database design decisions

## Skills Demonstrated
- Functional dependency analysis
- ERD interpretation
- Relational schema understanding
- Database theory application
- Analytical reasoning in data modeling
